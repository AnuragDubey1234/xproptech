# XPropTech.in - India's First PropTech Community

Where real estate meets technology, capital, and execution.

## Tech Stack

- **Next.js 15+** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- **Framer Motion**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push to GitHub
2. Import project in [Vercel](https://vercel.com)
3. Deploy — no config needed

Or use the Vercel CLI:

```bash
vercel
```
